This is a data analytics project focused on evaluating restaurant inspection results from New York City. Using inspection records from the NYC Department of Health, this project explores patterns in food safety violations, restaurant grades, cuisine types, and neighborhood performance.
The objective is to identify which restaurants and areas face the greatest food safety challenges, uncover the most common violations, and analyze trends across boroughs and over time. The insights generated can help city officials improve inspection strategies, prioritize high-risk locations, and develop targeted food safety education programs.
This project demonstrates skills in: data cleaning, exploratory data analysis (EDA), data visualization, trend analysis, and business-focused storytelling, transforming public health data into actionable recommendations for policymakers and stakeholders.
